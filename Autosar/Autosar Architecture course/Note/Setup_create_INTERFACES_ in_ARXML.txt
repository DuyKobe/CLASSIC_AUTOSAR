*
XMLSpy là một phần mềm phát triển XML và XSLT. Nó được phát triển bởi SyncroSoft và được sử dụng bởi các nhà phát triển phần mềm, kỹ sư hệ thống và nhà thiết kế XML.

XMLSpy cung cấp một loạt các tính năng cho phép người dùng tạo, chỉnh sửa, kiểm tra và debug các file XML. Nó cũng hỗ trợ các ngôn ngữ XSLT, XPath và XQuery.

* Create interfaces in ARXML
 + Copy cấu hình autosar và tạo tag closing </AUTOSAR>
 + Về <SENDER-RECEIVER-INTERFACE>: 
    + quan trong <SHORT-NAME> và <DATA-ELEMENTS>
      + <DATA-ELEMENTS> thì có <VARIABLE-DATA-PROTOTYPE>
        +<VARIABLE-DATA-PROTOTYPE> thì có <SHORT-NAME> và <IMPLEMENTATION-DATA-TYPE-REF>
 + Về <SENDER-RECEIVER-INTERFACE>: 
    + quan trong <SHORT-NAME> ,<OPERATIONS> -> <CLIENT-SERVER-OPERATION> và <POSSIBLE-ERRORS> -> <APPLICATION-ERROR>
      + <CLIENT-SERVER-OPERATION> thì có tên hàm <SHORT-NAME> và tham số hàm <ARGUMENTS> và <POSSIBLE-ERROR-REFS>

          +   <ARGUMENTS> có <ARGUMENT-DATA-PROTOTYPE>	  // thuộc tính data
			     <SHORT-NAME>FanSpeed</SHORT-NAME> //tên data 
			     <TYPE-TREF DEST="IMPLEMENTATION-DATA-TYPE">/AUTOSAR_PlatformTypes/ImplementationDataTypes/uint8</TYPE-TREF> //tham                       chiếu đến kiểu dữ liệu
 			     <DIRECTION>IN</DIRECTION> // input hay output hay input và output
          + <POSSIBLE-ERROR-REFS>  có <POSSIBLE-ERROR-REF DEST="APPLICATION-ERROR">/Interfaces/I_StartMotor/E_OK</POSSIBLE-ERROR-REF>
								<POSSIBLE-ERROR-REF DEST="APPLICATION-ERROR">/Interfaces/I_StartMotor/E_NOT_OK</POSSIBLE-ERROR-REF>

    + <POSSIBLE-ERROR>  có <SHORT-NAME>E_OK</SHORT-NAME>
			   <ERROR-CODE>0</ERROR-CODE>
                        và <SHORT-NAME>E_NOT_OK</SHORT-NAME>
			   <ERROR-CODE>1</ERROR-CODE>
